# Project Bletchley

> [!IMPORTANT]
> **DEVELOPMENT NOTICE:** This codebase and all associated laboratory workspaces are developed and maintained **exclusively via autonomous out-of-band coding agents** to establish an automated tinkering, prototyping, and experimentation environment.

Project Bletchley is a minimalist, neo-grotesque hub dashboard and playground suite scaffolded with the **Astro** framework, configured with **Tailwind CSS v4** and self-hosted variable fonts.

---

## 🛠️ System Architecture

The project is structured to maintain a clean separation of experiments and presentation pages under a modular `labs` directory:

```text
project-bletchley/
├── .github/
│   └── workflows/
│       └── deploy.yml          # GitHub Actions CI/CD to GitHub Pages
├── src/
│   ├── layouts/
│   │   └── Layout.astro        # Base layout with global CSS, theme switches
│   ├── pages/
│   │   ├── labs/
│   │   │   └── typocraft.astro # Typocraft Variable Font Playground
│   │   └── index.astro         # Minimalist Neo-grotesque Hub Dashboard
│   └── styles/
│       └── global.css          # Tailwind CSS v4 entry point & variable theme tokens
├── astro.config.mjs            # Astro config (Tailwind Vite Plugin & relative base)
└── package.json                # Project dependencies and script endpoints
```

## ⚙️ Core Specifications

1. **Light/Dark Switcher**: Responsive, cookie-free persistence theme switcher using CSS Variables and standard `data-theme` selectors. No flashing on load.
2. **Typography**: Native stack targeting the **Geist** and **Geist Mono** variable fonts from Fontsource for modern neo-grotesque aesthetics.
3. **Interactive Typography Playground (`/labs/typocraft`)**: Live canvas to test variable font axes (`wght`, size, tracking, leading) with real-time sliders and dynamic presets.
4. **Deployments**: Compiles statically and deploys automatically to GitHub Pages on every push to the `main` branch via GitHub Actions.

## 🚀 Commands & Development

Ensure you have Node.js v22+ installed on your host system.

```bash
# Install package dependencies
npm install

# Start the interactive development server
npm run dev

# Compile static assets for production
npm run build

# Preview production build locally
npm run preview
```
